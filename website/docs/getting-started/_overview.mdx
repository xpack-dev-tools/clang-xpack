The [clang](https://clang.llvm.org) project provides a language front-end and tooling infrastructure for languages in the C language family (C, C++, Objective C/C++, etc) for t[LLVM](https://llvm.org), a project maintained by the
[LLVM Foundation](https://foundation.llvm.org/).

**clang** is an open source project hosted on
[GitHub](https://github.com/llvm/llvm-project).
